Detailed Description

Stack frame (processor context) for ARM.

Context is stored on stack and it looks like this:

higher adresses: SPSR (CPSR form task) LR (PC - return address) IP FP R10 R9 R8 R7 R6 R5 R4 R3 R2 R1 R0 LR (task's LR) SP :lower adresses
