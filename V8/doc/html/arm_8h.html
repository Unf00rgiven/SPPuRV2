ARM processor modes.

This file contains macros for 8 different processor modes and one mask. Mode change is done by executing cps #mode instruction.

#define ARM_MODE_MASK   0x1f

Arm processor mode mask.

In order to get the processor mode, read the CPSR and apply this mask since last five bits contain processor mode.
