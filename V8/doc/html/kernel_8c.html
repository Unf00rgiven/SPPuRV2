◆ TaskDisp()

void TaskDisp ( struct STACK_FRAME *  stack)

Supervisor call (SVC) handler.

This function will be called every time SVC instruction is executed. Address of STACK_FRAME structure is passed as argument. All registers from current process are stored in that structure, and if task has to be switched, they are saved in memory and registers of next process are loaded. When this function finishes, registers from this structure are loaded back in CPU's registers.

Parameters
stackpointer to structure holding current task's registers
